Programming Projects

As a computer science student I have done many programming projects that I am proud of. Embedded is a small sample of them.

PutMeOn

This is my senior capstone project that I am working on with three classmates. We are attempting to create the infrastrucutre for a music based social media app. The backend (repo) is a Flask API using a noSQL Mongo cloud database. This project is also an opportunity to familiarize myself with various Agile practices like kanban, unit testing, continous integration with Travis and Github Actions and continous deployment with Heroku. The frontend (repo) is currently being built in React and being deployed to Netlify.

Learning the Structure of Biomedical Relationships from Unstructured Text

This is a project that I did with a group called STEM-Away. I was the technical lead for the project, which was to reproduce the research done in the titular 2015 research paper by Percha and Altman. The project involved many machine learning and NLP practices, including supervised and unsupervised biclustering, web scraping, XML parsing, and dependency parsing to cluster drug-gene relationships in over 23 million unorganized medical articles distributed across dozens of gigabytes of XML files. It was a great challenge to help coordinate such a project, especially given the fact that many group members were joining us from abroad and the entire project was done remotely, but I think we did a great job.

Fake News Filter

This was my final assignment submission for my introduction to machine learning class. I employed NLP to classify news articles from a kraggle database of over 45000 articles as fake or real news. The project has a thoroughly written Jupyter notebook as well as documentation to explain the process, but I tried support vector machines, neural networks, and logistic regression to find the best model for this task. The best model achieved a test accuracy greater than 97%.

Databases Demo Website

This project was my final submission for my introductory databases course in SQL. A partner and I collaborated to build a sample airline management system. We built the (relatively crude) frontend in HTML and CSS and used Pytorch and MySQL for the backend. The underlying SQL database uses 13 interconnected tables, and the project required 26 HTML pages built from scratch. It also employs data visualization using matplotlib, and is resistant to SQL injection attacks.
