Johnlouis Dahhan Week 12: Animate On Scroll and Particles.js
I experimented with 2 javascript libraries for this project. One was used to create this background and the other to create this scrolling animation.
Particles.js is a javascript library that allows for the creation of particle effects such as the ones in the background of this page. The actual javascript is incredibly light, as the configuration of particles is all done through a .json file that is loaded in the script.
The other library I used for this website is Animate On Scroll, which is incredibly easy to set up. All the javascript used was incorporating and initializing the library. The animation can be changed by simply changing an attribute of the div.
These libraries are interesting because despite the very interesting functionality they allow, they are incredibly easy to use, if slightly difficult to customize.
